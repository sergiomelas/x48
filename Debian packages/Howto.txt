download hp calculator rom rom from www.hpcalc.org
then (sx example)
$ wget https://www.hpcalc.org/hp48/pc/emulators/sxrom-j.zip
$ mkdir -p ~/.hp48
$ cp ./sxrom-j.zip ~/.hp48
$ cd ~/.hp48
$ bunzip2 sxrom-j.zip
$ mv sxrom-j rom
$ ln -s rom rom.dump
finally you need to initialize x48:

$ cd ~/.hp48 && x48 -initialize

Run x48 from the icon and clear memory
